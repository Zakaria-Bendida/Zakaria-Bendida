# 🚀 Main Projects

### 🚑 FEDZ — Fast Emergency Dispatch Algeria
Real-time web & mobile ambulance dispatch platform for Sidi Bel Abbès, combining live GPS tracking, blocked-road management, and intelligent Dijkstra-based routing with fast and fragile-patient comfort modes.
- PostgreSQL · PostGIS · pgRouting
- Node.js · Express · Socket.IO
- React · OpenLayers
- React Native · Expo

---

### 🛂 Saudi eVisa — Digital Visa Request Platform
Full-stack MERN application that digitizes a multi-page visa application form into a guided, multi-step web experience with real-time validation and file upload.
- React · Vite
- Node.js · Express
- MongoDB · REST API
